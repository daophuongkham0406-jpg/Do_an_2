Cấu hình

Cài đặt API

Ví dụ: https://your-server.com/api hoặc http://localhost:3000/api

MongoDB Schema mẫu

{
  _id: ObjectId,          // MongoDB tự tạo
  name: String,           // Tên bài tập (bắt buộc)
  muscle: String,         // Nhóm cơ chính
  icon: String,           // Emoji icon
  diff: "B"|"I"|"A",     // Độ khó
  equip: String,          // Thiết bị
  sets: String,           // Số sets
  reps: String,           // Số reps
  rest: String,           // Thời gian nghỉ
  sec: [String],          // Cơ phụ (mảng)
  steps: [{               // Hướng dẫn (mảng)
    t: String,            //   Tiêu đề bước
    d: String             //   Mô tả bước
  }],
  tips: [String],         // Lưu ý (mảng)
  createdAt: Date,
  updatedAt: Date
}

API Endpoints cần có

GET /api/exercises Lấy toàn bộ bài tập
POST /api/exercises Thêm bài tập mới
PUT /api/exercises/:id Sửa bài tập
DELETE /api/exercises/:id Xóa bài tập
